# Assets Directory

This directory contains the base portfolio tracker application file that can be used as a starting point for modifications.

## Files

### portfolio-tracker-base.html
The complete single-file portfolio tracking application with all features implemented:
- Month-specific data persistence
- Drag-and-drop row reordering
- Cell editing with formatting (bold, larger font)
- Auto color-coding (green/red for positive/negative)
- Calculated Standing AT row
- UnRealized and Realized P&L rows
- Local storage persistence
- Notes section
- Responsive design
- Financial terminal aesthetic

## Usage

To use this as a starting point:
1. Copy this file to the working directory
2. Make modifications as needed
3. Test in browser
4. Provide to user via present_files tool

To extend functionality:
1. Read the implementation.md reference file for patterns
2. Modify the appropriate sections (see SKILL.md for guidance)
3. Test thoroughly, especially drag-and-drop and month switching
4. Ensure data persistence works correctly
